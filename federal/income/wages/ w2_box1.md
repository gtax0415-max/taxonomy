# Form W-2
## Description
Employer wage reporting document.
## Required Information
- Employer name
- EIN
- Wages
- Federal withholding
## Questions
- Did you receive a W-2?
- Did you have more than one employer?
## Common Errors
- Missing W-2
- Wrong SSN
